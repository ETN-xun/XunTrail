# 隐私政策 / Privacy Policy

## XunTrail 埙音游戏

**最后更新日期：** 2024年12月

---

## 1. 概述

XunTrail（埙音游戏）是一款单机音乐游戏，致力于保护用户的隐私和数据安全。本隐私政策说明我们如何收集、使用和保护您的信息。

## 2. 信息收集

### 2.1 我们不收集的信息
- **个人身份信息**：我们不收集您的姓名、邮箱、电话号码或其他个人身份信息
- **网络数据**：游戏完全离线运行，不向任何服务器发送数据
- **设备信息**：我们不收集您的设备标识符、IP地址或位置信息
- **使用统计**：我们不收集游戏使用统计或分析数据

### 2.2 本地数据存储
游戏仅在您的设备上本地存储以下数据：
- **游戏设置**：音量、按键配置等用户偏好设置
- **游戏进度**：分数记录和挑战模式进度（如有）
- **乐谱文件**：您选择加载的本地乐谱文件

## 3. 数据使用

### 3.1 本地数据用途
- 保存您的游戏设置和偏好
- 记录游戏进度和成绩
- 提供个性化的游戏体验

### 3.2 文件访问
- 游戏会访问您选择的乐谱文件（.txt格式）以提供游戏内容
- 游戏会读取StreamingAssets文件夹中的默认乐谱文件
- 所有文件访问都需要您的明确操作或同意

## 4. 数据安全

### 4.1 本地存储安全
- 所有数据仅存储在您的本地设备上
- 没有数据传输到外部服务器
- 您可以随时删除游戏来清除所有相关数据

### 4.2 文件安全
- 游戏仅读取您明确选择的乐谱文件
- 不会修改或删除您的原始文件
- 不会访问系统中的其他文件

## 5. 第三方服务

本游戏不集成任何第三方分析、广告或数据收集服务。

## 6. 儿童隐私

本游戏适合所有年龄段用户使用，不专门针对儿童设计，也不会故意收集儿童的个人信息。

## 7. 数据控制

### 7.1 您的权利
- **访问权**：您可以随时查看游戏存储的本地数据
- **删除权**：您可以通过卸载游戏来删除所有相关数据
- **控制权**：您完全控制要加载的乐谱文件

### 7.2 数据删除
要完全删除游戏相关数据：
1. 卸载游戏应用程序
2. 删除游戏安装目录（如需要）
3. 清理系统注册表中的相关条目（Windows系统）

## 8. 隐私政策更新

我们可能会不定期更新此隐私政策。任何重大变更都会在游戏中或通过其他适当方式通知用户。

## 9. 联系我们

如果您对此隐私政策有任何疑问或建议，请通过以下方式联系我们：

- 项目地址：[GitHub Repository]
- 问题反馈：通过GitHub Issues提交

## English Version

### Overview
XunTrail is a standalone music game that prioritizes user privacy and data security. This privacy policy explains how we handle your information.

### Information We Don't Collect
- Personal identification information
- Network data (the game runs completely offline)
- Device identifiers or location data
- Usage analytics or statistics

### Local Data Storage
The game only stores data locally on your device:
- Game settings and preferences
- Game progress and scores
- Music sheet files you choose to load

### Data Security
- All data remains on your local device
- No data transmission to external servers
- You can delete all data by uninstalling the game

### Contact
For questions about this privacy policy, please contact us through our GitHub repository.
